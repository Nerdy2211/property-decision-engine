# UI Notes — Property Decision Engine

## Goal
Create a clean, dark-themed Streamlit app that feels modern, premium, and investor-oriented.

## Style
- dark theme
- clean cards
- teal / cyan accent
- report-like layout
- modern, minimal, professional
- not flashy

## Page 1 Layout
Top:
- title
- Australia Property Climate Score card
- short market summary

Below:
- factor score cards
- city score cards
- charts
- explanation panel
- risks vs tailwinds section

## UX Principles
- prioritize clarity
- make charts readable
- keep sections well separated
- use expanders for detailed reasoning
- keep language practical, not academic

## Important
This is a decision-support dashboard, not a generic blog page.
It should feel like a premium market intelligence tool.
